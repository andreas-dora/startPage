Time Conversion

Bei einer Übung mit der Open Weathermap-IDE blieb ich mal wieder an einer so scheinbar einfachen Nebensache hängen. Die Zeiten für Sonnenauf- und Untergang werden in UTC-Timestamps ausgegeben. Die kennt man ja vielleicht schon aus der Tabellenkalkulation. Dort gab es nie Probleme damit. Nachdem ich an den gesammelten Javascript Date-Methoden verzweifelt bin, kam ich zu folgender Lösung 


  function stampConverter(utcStamp){

    var date = new Date(utcStamp * 1000);
    var h = date.getHours();
    var m = "0" + date.getMinutes();
    var seconds = "0" + date.getSeconds();
    var myTime = h + ':' + m.substr(-2) + ':' + seconds.substr(-2);
    
    return myTime;
  };
